# Privacy Notice / 隐私说明

生效日期：2026-07-15

## 当前版本收集什么

Physics Visual Lab v5.0.0 本身不设置账号，不投放广告，不安装分析脚本，也不建立访问者 IP、设备指纹或个人资料数据库。

网站会在访问者自己的浏览器中使用 `localStorage` 保存：

- 手动选择的界面语言；
- 日间或夜间主题；
- 电路工作台的本地草稿。

这些数据用于恢复界面状态，不会由本站代码上传给维护者。访问者可以在浏览器站点设置中随时清除。

## 托管平台

网站由 GitHub Pages 提供静态托管。GitHub 可能为安全、运维和服务交付处理请求日志，包括网络与设备信息；这些处理受 [GitHub General Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) 约束。本站维护者目前不能通过本网站读取 GitHub 的原始访问日志，也不会据此识别个人访问者。

## 外部内容

当前应用运行所需的脚本、样式和字体均随站点提供，不需要从第三方字体或分析服务加载。页面中指向文献、GitHub 或其他网站的链接受目标网站自己的隐私规则约束。

## 将来如果加入统计

在启用任何访问统计、错误追踪、邮件订阅或付费功能前，维护者应先：

1. 明确说明收集字段、目的、法律依据、处理方、保留期限与删除方式；
2. 默认采用聚合统计和 IP 截断，避免保存原始 IP 与设备指纹；
3. 在适用地区提供同意、拒绝或退出机制；
4. 签署所需的数据处理协议，并评估跨境传输；
5. 更新本说明后再上线功能。
